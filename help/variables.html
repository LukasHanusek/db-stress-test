Variables

How to use Variables in SQL

To use variables in the SQL sets, we use '?' symbol.
Example: SELECT * FROM TAB WHERE ID=? AND AMOUNT > ?

Variables are declared right below the SQL text area.


Variables syntax

  • If we use more than 1 variable, variables are split by using "Argument delimiter", by default represnted by ',' character.
  • Variables sets are delimited by "Line delimiter", by default represented by ';' character.
  • SQL query is executed more times if more variables sets are declared and delimited by "Line delimiter"
