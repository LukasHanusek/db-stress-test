Funtion randomInt(MIN,MAX)



#{randomInt(int min, int max)}
Function generates a random integer number

Parameters:
min - minimum (requiered)
max - maximum (requiered)

Returns:
Random Integer number within the specified range

Throws:
InvalidFunctionException - if arguments are not integer numbers

Example:
#{randomInt(1,10)} - generates a random number from 1 to 10
