Funtion randomCharEnum(enumName)



#{randomCharEnum(String enumName)}
Function generates a random enumeration of characters from the given pre-defined character enumeration in "CharEnum" menu.

Parameters:
enumName - pre-defined character enumeration name (requiered)

Returns:
Randmized character string of the same length as the pre-defined character enumeration

Throws:
InvalidFunctionException - if the given character enum does not exist during function initialization
FunctionEvaluationException - if the given character enum is empty or non-existent during the function execution

Example:
#{randomCharEnum(A)} - generates a random string of characters from the 'A' named character enumeration.
