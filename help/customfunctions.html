Custom functions

How to use custom functions

To use functions we need to use variables in the SQL sets as described in Using variables.
Example SQL: SELECT * FROM TAB WHERE ID=? AND AMOUNT > ?

Custom functions are declared in the variables area right below SQL.


Functions syntax

  • All functions start with # symbol and are encapsulated in between { and } parentheses. Example: randomInt(1,10)
  • Functions are not alowed in combination with varchar variables or any others.
  • It is not possible to use function as argument to another function.
